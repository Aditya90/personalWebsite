Robockey

Course : MEAM 510 - Design of Mechatronic Systems
Professor : Dr. Jonathan Fiene

Description

As a team, we built 3 autonomous robots which had to localise based on an overhead IR constellation, move inside a custom rink and manipulate a puck with the aim of playing hockey. The bots had to be designed from the ground up, including the structure and all on board electronics and code.

Personal Objectives

  • Designed and assembled all electronics required for interfacing sensors and motors with the processor
  • Designed and assembled an analog circuit for detecting if the puck has been acquired by the robot
  • Made a motor actuated kicking mechanism
  • Supplemented the onboard code for game play, wireless command reception and robot motion
