Description

The aim of this project is to develop an autonomous garden watering system. The system should adhere to the following requirements :

  1. Be compact enough to fit nicely in a small apartment
  2. Operation off of a rechargable battery
  3. Be able to provide sufficient water to all the plants being monitored
  4. Detect and log information from sensors to know when to water the plants
I have also hosted this project on Hackster IO.
