Logo Replacement using Matlab

Course : CIS 581 - Computer Vision and Computational Photography
Professor : Dr. Jianbo Shi

Description

The objective of this project was to take a target logo and a replacement logo, and to replace the target logo in several images with the replacement logo using a Matlab Script.

Tasks

  • Detected instances of the target logo in multiple images succesfully (succes rate of > 70 %)
  • Performed Laplacian Blending while inserting the replacement logo to avoid prominent boundaries
