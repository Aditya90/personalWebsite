ROS Robot

Description

This project is aimed at creating a robot based on the Beagle Bone Black using the ROS architecture. The motivation behind this project is to create a software package which is modular so that individual components (like motor drivers) can be easily replaced. I have hosted this project on Hackster IO and you can check there for more information.
